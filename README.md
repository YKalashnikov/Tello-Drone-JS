# Tello-Drone-JS
Tello Drone Java Script inspired by WESBOSS


![](https://github.com/YKalashnikov/Tello-Drone-JS/blob/master/screenShot.png)

### Using
- ReactHooks
- NodeJS
- Socket.io
- Styled-components
- Waait
- Express
- Nodemo
-Next.js

```shell
-frontend
$ npm install
$ npm run dev
$ go to http://localhost:7777
```

```shell
-backend
$ npm install
$ npm run dev
$ go to http://localhost:7777
$ node drone.js
```
